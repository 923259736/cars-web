##### git

```shell
Sourcetree

//拉取仓库所有分支
git branch -r | grep -v '\->' | while read remote; do git branch --track "${remote#origin/}" "$remote"; done
git fetch --all
git pull --all

创建分支 ：git branch <name:分支名称>
切换分支 ：git checkout <name:分支名称>
创建并切换分支 ：git checkout -b <name:分支名称>
合并分支： git merge --no-ff -m '合并分支说明' <合并分支的名称>
```
